## ✨ Features

- **AI Career Coaching** — Get personalized career advice and growth plans
- **Skill Gap Analysis** — Identify missing skills for your target roles
- **Resume Optimizer** — AI-powered resume analysis and improvement suggestions
- **Job Matching** — Smart recommendations based on your profile
- **Interview Preparation** — Practice common questions and get feedback
- **Career Roadmaps** — Step-by-step paths to your dream job
- **Progress Tracking** — Monitor your career development over time

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/Dastagiri3/career-compass-ai-41.git
cd career-compass-ai-41

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your API keys (OpenAI, etc.)

# Run the development server
npm run dev
